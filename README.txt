1.카메라 설정 관련
(a) : 초기 카메라가 설정 (250, 250, 0)에서 바라봄
(b) : 1, 2, 3, 4,모두 구현

2. 기하물체 관련
(a) 가운데 마리오와 주변에 오토바이를 배치
(b) 비행기 3대를 계층적 모델링 하여 배치
(c) 오토바이와 비행기가 마리오 주변을 돌면서 자전함 + 메뉴버튼에 Add bike를 누르면 오토바이가 하나씩 증가

3. 광원설정

(a) 세상 좌표계 기준으로 마리오 위에서 (0, 100, 0)에서 바닥으로 광원 설정
(b) 모델링 좌표계 기준으로 비행기 위에서 붉은색 광원 설정
(c) 눈 좌표계 기준으로 카메라에서 바라보는 방향으로 광원 설정 - 해당 광원은 평행광원임
(d) 눈 좌표계 기준으로 카메라에서 바라보는 방향으로 스폿광원 설정 - 메뉴 버튼에 blind effect를 누르면 비행기 위 조명에 블라인드 효과 생성


4. 텍스쳐 설정 관련
매뉴버튼에 My Textrue effect를 누르면 바닥이 풀 -> 눈 -> 사막으로 바뀜
메뉴버튼에 Texture on/off 를 누르면 텍스쳐 매핑을 on/off 할수 있음

5. 쉐이더 관련
(a) 기본 퐁 쉐이딩으로 구현
(b) 매뉴버튼에 Fog effect를 누르면 안개효과 생성 / 메뉴 버튼에 blind effect를 누르면 비행기 위 조명에 블라인드 효과 생성
